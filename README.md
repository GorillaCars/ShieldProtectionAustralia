# Shield Protection Australia

Premium static landing page and forms page for Shield Protection Australia.

## Forms

Place or replace the final PDFs in `forms/downloads` using these filenames:

- `PRODUCT APPLICATION FORM _ SCHEDULE.pdf`
- `Form.pdf`
- `Authority Form.pdf`
- `Service Coupons.pdf`
- `Claim Checklist.pdf`

The Forms page download buttons are already pointed at those files.

## Supabase

The site is pre-wired with the public Supabase project URL and publishable key in `supabase-client.js`.
